# 📊 E-Commerce Sales Data Analysis

## 📌 Project Overview

This project focuses on analyzing e-commerce sales data to extract meaningful business insights using Python(Pandas and Matplotlib). The goal is to understand sales trends, product performance, and regional distribution to support data-driven decision-making.

---

## 🎯 Business Problem

Businesses often struggle to identify which products, regions, and time periods contribute the most to revenue. This project aims to solve that by analyzing historical sales data and uncovering actionable insights.

---

## 🎯 Objectives

* Analyze monthly sales trends
* Identify top-performing products
* Evaluate region-wise performance
* Understand category-wise contribution to revenue

---

## 🛠 Tools & Technologies

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

---

## 📂 Dataset

The dataset contains the following columns:

* Order_ID
* Order_Date
* Region
* Product
* Category
* Quantity
* Price

---

## ⚙️ Project Workflow

1. Data Collection
2. Data Cleaning & Preprocessing
3. Feature Engineering (Revenue, Month extraction)
4. Exploratory Data Analysis (EDA)
5. Data Visualization using Matplotlib
6. Insight Generation

---

## 📊 Key Insights

* Electronics category generated the highest revenue
* Certain regions consistently outperformed others
* A few products contributed significantly to overall sales
* Monthly trends revealed variations in sales performance
